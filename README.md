# Work Timer Python

A simple work timer GUI in Python using Tkinter. Tracks work sessions and logs them to a text file.

## Features
- Start/Stop timer
- Logs work sessions to `work_log.txt`
- Displays current date and live timer
- Shows total worked time on stop

## Requirements
- Python 3.x
- Tkinter (usually included with Python)
